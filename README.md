# Politeia Book Canonical Scraper

A collection of web scrapers for different book-related websites.

## Project Structure

Each website has its own dedicated folder containing:
- Scraper scripts
- Configuration files
- Output data
- Documentation specific to that website

## Websites

### ISBN Chile (`isbnchile/`)
- **URL**: https://isbnchile.cl/
- **Purpose**: Scrape book data from the Chilean ISBN registry
- **Status**: In development

## Setup

```bash
# Install dependencies
pip install -r requirements.txt
```

## Usage

Navigate to the specific website folder and follow the instructions in its README.

## Requirements

- Python 3.8+
- See `requirements.txt` for package dependencies
